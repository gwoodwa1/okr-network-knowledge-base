# Network Engineering Knowledge Bundle

An illustrative OKF repository derived from the documents in `source-docs/`. It demonstrates a navigable hierarchy, typed concept documents, bundle-relative links, and a chronological update log.

# Knowledge areas

* [Designs](designs/) - High- and low-level designs for data-center and core networks.
* [Standards](standards/) - Baseline engineering and change-control conventions.
* [Operations](operations/) - Procedures for troubleshooting, onboarding, recovery, and maintenance.
* [Examples](examples/) - Illustrative network configuration snippets.
* [APIs](apis/) - REST API integration guidance.

# Bundle information

* [Update log](log.md) - Chronological changes to this bundle.
* This bundle is fictional and intended for OKF demonstrations, not production deployment.
